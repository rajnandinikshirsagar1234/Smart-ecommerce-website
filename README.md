 Smart E-Commerce Website

##  Project Overview

This project is a **Smart E-Commerce Website** developed using **HTML, CSS, and JavaScript**.
It simulates a basic online shopping platform where users can browse products and manage their cart efficiently.

##  Outputs

* Home Page

Home Page-(images/home.png)

* Cart Page

Cart Page-(images/cart.png)


## Features

- Browse products on the home page
- Add items to cart
- View selected items in cart page
- Clean and attractive user interface
- Interactive functionality using JavaScript


## Technologies Used

HTML 
CSS 
JavaScript
